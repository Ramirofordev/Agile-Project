# Agile Task Manager

## Project Overview

This project is a Task Management System developed using Agile methodology (Scrum).
The goal is to build a simple and scalable web application that allows users to manage tasks efficiently.

The project follows a layered architecture and version control best pracatices using Git.

---

## Objectives

- Apply Agile (Scrum) methodology
- Implement user stories through sprints
- Follow clean architecture principles 
- Maintain proper documentation
- Use Git with structured commites and branches

---

## Techonologies Used

- Python
- Flask
- SQlite
- HTML / CSS
- Git & GitHub

---

## Project Strucutre

- domain/ -> Business Model
- infastructure/ -> Database & repositories
- services/ -> Business logic layer
- templates/ -> HTML views
- static/ -> CSS / JS
- docs/ -> Agile documentation

## Agile metodology

This project follows Scrum: 

- Product Backlog
- Sprint Planning
- User stories
- Definition of Done
- Incremental Development

1. Create virtual environment
2. Install Dependencies
